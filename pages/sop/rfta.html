Request for Tenancy Approval (RFTA)

In order for a client's prospective unit to be inspected, the request for tenancy approval must be filled out and completed by the client and prospective property owner.

  1. Check RFTA to ensure it is complete and both the client and property owner have signed it. If any portion of the RFTA is left blank, follow-up with both parties before moving forward. Date stamp using the date the RFTA is completed and signed only.
  2. Property owner and/or manager must also submit the W9 tax form for all RFTAs submitted. All new W9s submitted on or after November 16, 2018 must use the October 2018 version. Any prior versions submitted are considered outdated and will not be accepted.
  3. The direct deposit enrollment form and voided check must also be filled out and accompanied by verification from the p.o. financial institution that confirms the account and routing numbers. A voided check that notes this information is preferred. This is only mandatory if the either the prospective property owner is new to the program or is looking to update their bank information.
  4. Complete the property owner data entry cover sheet consistent with information noted on the W9 tax form. Print this out, attach the W9 and direct deposit information, and submit to the leased housing data coordinator, the staff person responsible for verifying this information with the IRS. This is typically verified within 24 hours. If the verification is rejected, follow-up with the property owner/manager before proceeding
  5. Contact client and property owner to confirm this is all received. Place a priority one note and update the intake master list.
  6. Leave client's file in the intake coordinator's drawer.
  1. Begin processing on Tracker only if the W9 verification is approved.
  2. Run action "6.100 RFTA Submitted". The effective date will be the date the RFTA was received (which should already be date stamped) regardless of when it's being processed.

    The OTC tile will automatically remove the expiration date as the client's voucher is frozen from the date the RFTA is submitted to either the date the client leases-up or if the prospective unit is no-go'd.
  3. Click on the "Prospective Units" link under the "Actions" drop-down menu.
  4. Check to see if unit requested already exists in Tracker. If it does then ensure all of the relevant unit information is consistent with the RFTA. Update if necessary.
  5. If the unit does not exist in Tracker, change the "Curren Unit" option to "Add New Unit" option. Enter the unit information consistent with what is noted on the RFTA.

    The owner's information can be entered either by typing in the entity name noted on the verification or the ID number the leased housing data coordinator notes. Be sure this is noted in the owner, agent, and payee fields.
  6. Ensure client's family composition and income information is properly entered and updated.
  7. Click the "Calc 40 Percent" link in the "Actions" drop-down menu and then click on the "Utilities" tile.
  8. Select "Yes" and use chart "55" if the client is responsible for any of the utilities as reported on the RFTA. Save before proceeding.
  9. Update utilities only noting what the client will be responsible for based on what's reported on the RFTA. Save this and then click on "Calc Utility Allowance" to ensure the correct utility allowance is noted.
  10. Go back to the initial "Calc 40 Percent" then enter the rent amount that was requested on the RFTA. Ensure that the correct payment standard is noted.

    If the payment standard is inaccurate, manually enter the correct amount    . Calculate the maximum housing cost and note it on the RFTA.
  11. Go back to "Prospective Units" and click on the "Print Request Inspection - RFTA Form" button. Use the date the client intends to lease-up for the print the RFTA.

    Note the maximum housing cost and program type on the printed RFTA. Create a copy of the original RFTA and the printed one generated from Tracker and leave this in the file bound with W9 and direct deposit information.
  12. Submit original RFTA and the printed one generated from Tracker to either the inspection coordinator responsible for scheduling new unit inspections or the property owner services manager for RFTAs that include any de-leading verifications.
  13. Click here to email the inspection coordinator to confirm the unit was submitted. If the RFTA includes any de-leading documentation, click here to email the Property Owner Services Manager
  14. Contact the client and property owner to confirm the RFTA was submitted.
  15. Update the intake tracking sheet, leave a priority one note that documents it was submitted, and place the file in the intake coordinator's drawer.
