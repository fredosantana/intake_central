In-State Transfers

If a client requests a unit outside of Metro Housing|Boston's coverage area, the client may be eligible to transfer in-state.

  1. The client will need to submit a completed and signed RFTA along with the required property owner documents.
  2. Before moving forward, check which program the client is allocated under. If the client came off the Section 8 wait list, then proceed with the next steps. Otherwise check in with the intake coordinator or the program manager to confirm if the program the client is allocated under is eligible for a transfer out.
  3. Contact the client for updated income information consistent with the DHCD adminstrative plan. Regional agencies will not accept outdated income information even if it's the same as it was originally reported by the client.
  4. Once the updated income information is received, pull all documents from the file noted on the applicant transfer checklist.
  5. Create a new voucher with the issued date being the date the RFTA was received. The copy of the original voucher still needs to be included in the transferred file.
  6. Update income information if necessary. Then run a "Calc 10" with the effective date being the date of the new voucher created in the previous step. Print the 50058 as this needs to be included in the transferred file.
  7. Check the agency listing chart to confirm which agencies covers the city the client prospective new unit is in.
  8. Create the transfer courtesy letters that are consistent with the agency the file is being transferred to.
  9. Once the file is ready, scan it and then make a copy of it. The copy will remain in the client file while the original will be mailed to the regional agency.
  10. Email the file to the intake coordinator or the staff person covering in-state transfers. This staff person will handle all of the necessary Tracker steps.
  11. Contact the client and property owner/manager to confirm once the staff person has confirmed by email to the regional agency that the file was sucessfully transferred. Place a priority one note and update the intake tracking sheet.
  12. The file can be returned to the file room in the "Transfers" section of the intake area.
