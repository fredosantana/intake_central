Extension Requests

If a client has exhausted their first 60 days, they are entitled to a one-time 30 day extension. This must be a written request either directly from the client or their certifed service provider if the client was issued under a special program.

  1. Check "Waitlist History" tile to confirm client has not already received their first extension. If the client has already received their first extension, advise them of reasonable accomodation procedure if applicable
  2. After the first step is confirmed, click on "Perform Action" then select the client's appropriate program.
  3. Select the "6.001 1st 30 Day Extension" action. The effective date is two days after the date the action is performed as this allows the clock to be effective for the presumed date the client receives it by mail.
  4. Running the action will generate two courtesy letters with one to be included with the re-issued voucher packet and the other to go in the file.
  5. The OTC tile should note the new expiration date. If this is inaccurate please let the intake coordinator or program manager know. Either will fix this and will follow-up when completed.
  1. Create a new voucher and fill-in the voucher bedroom size, original issuance and expiration date, and client's name.
  2. Fill in the "Date Voucher Extension Expires" box with the updated expiration date. Print it.
  3. Print the most recent RFTA.
  4. The intake coordinator should sign the voucher once it's printed. If the intake coordinator is unavailable to sign it, please see a program manager as a Section 8 trained staff person must sign off on the voucher.
  5. The original signed voucher, courtesy letter, and RFTA are to be mailed to the client. Copy of the voucher and courtesy letter go in the file bound to the original written reqeust.
  6. Update the tracking sheet, leave a priority one note acknowledging the request was taken care of, and contact the client to confirm. The file can go back in the file room.
