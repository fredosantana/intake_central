Reasonable Accommodation Requests

A client may request a reasonable accommodation if they are seeking an exception to regulations noted in the DHCD adminstrative plan. The client or a member of their family composition must be either elderly or disabled to qualify.

  1. The front page of reasonable accommodation request form must be completed. While it is suggested the client fills this out, a staff person or any individual/organization allowed to speak on behalf of the client can fill this out.
  2. Once the form is completed, get the client's file and drop it off in the director of leased housing's reasonable accommodation box.
  3. Click here to email the fair housing department and the leased housing director to confirm the file's location. Place a priority one note in Tracker and update the intake spreadsheet. They will decide if the request is approved. They will also follow-up with the client directly if any additional documentation/information is necessary.
  4. Contact client to confirm the request was submitted appropriately and inform them that reasonable accommodation requests typically take up to seven business days to review.
  5. Click "Calc". Select "10 - Issuance of Voucher". The effective date will be the date the voucher was issued.
  6. After running calc, print the first page of the 50058. Leave the print out in the file.
  7. Update the tracking sheet to reflect clock dates and file location.
  8. Proceed with any follow-up noted on the reasonable accommodation determination form once the file is returned to intake.

Once a decision is made, then the client will be sent a determination letter. A copy will be included with the file when the file is returned to intake. Any follow-up will depend on the outcome of the reasonable accommodation. The two most common decisions intake follow-ups with or voucher extensions and adjusted bedroom size.

  1. If the request was denied, there is no follow-up needed from intake. Fair housing will leave a priority one note. The file would need to be placed back in the file room. Update the intake spreadsheet appropriately.
  2. If the request was approved, then follow-up appropriately based on the decision noted. Follow-up must be done within 24 hours after the file has been returned to intake.
  3. Voucher extension approval process in Tracker is similar to extensions. The action will be "6.____ Reasonable Accomodation Extension".
  4. Bedroom size change process in Tracker is similar to voucher re-issuance.
  5. Place a priority one note specifying the reasonable accommodation, update the intake spreadsheet, and return file to the file room.
